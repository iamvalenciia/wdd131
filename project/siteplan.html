Website Planning Document

Juan Valencia - Software Developer Portfolio

Site Name

Juan Valencia | Software Developer

This name directly communicates my personal brand as a software developer. It clearly indicates that this is a professional portfolio site for Juan Valencia, focusing on software development services and showcasing my work.

Optional domain: juanvalencia.dev

Site Purpose

The site serves as a professional portfolio to highlight my skills, experiences, and projects as a software developer. It aims to:

  • Showcase my technical skills and expertise in software development
  • Present my professional work experience and educational background
  • Display projects I've worked on with descriptions and technical details
  • Provide a way for potential employers or clients to contact me
  • Establish my personal brand in the tech industry

Scenarios

These are questions that potential visitors might have when visiting my site:

  1. Scenario 1: What experience does Juan have with software development and QA testing?
  2. Scenario 2: What programming languages and technologies is Juan proficient in?
  3. Scenario 3: How can I contact Juan for a potential job opportunity or collaboration?
  4. Scenario 4: What projects has Juan worked on and what were his contributions?

Color Schema

The color scheme is designed to be professional, clean, and modern, reflecting my identity as a software developer:

Primary
#3498db
Secondary
#2c3e50

Color Usage

  • Primary Color (#3498db): Used for buttons, links, highlights, and accent elements to draw attention to important information.
  • Secondary Color (#2c3e50): Used for headers, footers, and backgrounds to create a professional contrast.
  • Light Color (#ecf0f1): Used for page backgrounds to ensure readability.
  • Text Color (#333): Used for body text to ensure readability.

Typography

The typography is chosen to be clean, professional, and highly readable:

Primary Font: Segoe UI

This is the main font used throughout the site for all text elements. It's a clean, modern sans-serif font that's highly readable on screens.

Usage:

  • Headings: Bold weight, slightly larger size
  • Body text: Regular weight
  • Buttons and interactive elements: Semi-bold weight
